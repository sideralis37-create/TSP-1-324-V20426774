Trabajo sustitutivo de prueba de la unidad curricular Computación II 324 de la Universidad Nacional Abierta UNA.
Unidad 1, TAD usando clases para calcular la suma de 2 números complejos.
Unidad 2, Lista Enlazada para calcular el máximo y mínimo elemento de una lista de n números 
enteros.
Unidad 3, uso de lista enlazada para calcular la suma de una lista de n números enteros y su promedio.
